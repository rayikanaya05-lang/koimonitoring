Pengertian

Dashboard ini digunakan untuk memantau parameter lingkungan secara real-time mencakup pH air, suhu air, dan kualitas air guna memastikan pertumbuhan ikan koi optimal.

KOMPONEN

Mikrokontroler:

1. ESP32 Dev Kit

2. Sensor pH: untuk mengukur tingkat keasaman (pH 7-8.5 ideal).

3. Sensor Turbidity: untuk mengukur kejernihan air (kekeruhan).

4. Sensor Suhu: untuk memantau suhu air kolam (ideal 20°C-30°C)

Aktuator:

1. Pompa Air Kecil (Dosing Pump): Untuk meneteskan ekstrak daun ketapang.

2. Motor Servo: Sebagai opsi mekanis untuk membuka/menutup jalur makanan/palet.

Lainnya: Relay Module (untuk pompa), Kabel Jumper dsb.
